# Roles + Permissions

## `@everyone`

- View Channels
- Embed Links
- Add Reactions
- Use External Emoji
- Use External Sticker
- Read Message History
- Use Application Commands
- Use Voice Activity

## `@core`

Special role just given to special people.
Inherit permissions from @everyone.

## `@admin`

Special role just given to myself.
Inherit permissions from @everyone.

## `OURSPACE` Category Overrides

For `@everyone`:

- Send Messages
- Connect
- Speak
- Video
